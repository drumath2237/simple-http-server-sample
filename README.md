# simple-http-server-sample

## Install

```bash
yarn install
```

## Usage

### Server side

サーバーを立ち上げた後にjsonデータをPostすると
tsst.jsonに内容を保存します。

簡単なクライアントのサンプルも作ってあります。

```bash
yarn server
# or
# yarn ts-node index.ts
```

### Client side test

```bash
yarn request
# or 
# yarn ts-node client.ts
```
